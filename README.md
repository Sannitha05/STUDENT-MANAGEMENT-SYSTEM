# STUDENT-MANAGEMENT-SYSTEM
simply - PARENTS COULD ACCESS THE STUDENT'S SEMESTER MARKS, GRADES.

The Student Management System is a web-based application designed to help educational institutions manage student academic information efficiently. This system allows administrators to upload semester marks, while students and parents can securely view academic results online.

## Project Overview
This project simplifies the process of storing and accessing student academic records. It provides a user-friendly interface for managing student data, reducing manual paperwork and improving transparency between institutions, students, and parents.

## Features
- Student Registration System
- Secure Login Authentication
- Admin Panel for uploading student marks
- Students and parents can view semester marks and grades
- Simple and user-friendly interface
